What is LastLife

LastLife is a Minecraft SMP server in the same style as the official LastLife Server (no, I did not steal the name, we came up with this name while it was still called ThirdLife). Players are randomly given 2 to 6 lives, and when on their final life they are allowed to kill whoever they choose. Players on lives above can only fight in self defense. In LastLife, almost anything within the game goes. This includes griefing, looting, trapping, the only exception being hacking.

The server is played in sessions, to ensure that almost all players are online at the same time. Outside these sessions, the server is not online.

Why does this need a launcher?

I'm sick of teaching people how to install mods, and since LastLife uses this voicechat mod I figured it would be easier to make a launcher.

Is there a boogeyman?

I'm not planning on it, I think adding luck to the server makes it less enjoyable. However, if the majority wants it, it is already part of the plugin and will be enabled.
